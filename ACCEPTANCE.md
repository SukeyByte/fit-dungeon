# FitDungeon 验收标准

## v0.1 - 原型验证

### 功能验收
- [ ] Flutter 项目能成功编译运行
- [ ] 摄像头权限申请正常
- [ ] 摄像头画面能显示在屏幕上
- [ ] MediaPipe/ML Kit 能识别人体关键点
- [ ] 深蹲动作识别准确率 > 70%
- [ ] 像素小人角色能显示
- [ ] 做深蹲时小人能做出攻击动作

### 测试方法
1. 安装 APK 到 Android 手机
2. 打开 App，允许摄像头权限
3. 站在摄像头前，做深蹲动作
4. 验证小人是否同步做出攻击动作

### 交付物
- `fit-dungeon-v0.1.apk`
- 源代码
- 测试视频

## 验收流程

1. **开发完成** → 自测
2. **自测通过** → 打包 APK
3. **打包完成** → 上传 GitHub Release
4. **上传完成** → 更新 PROGRESS.md
5. **通知苏柯柯** → 用户验收
6. **验收通过** → 进入下一版本
